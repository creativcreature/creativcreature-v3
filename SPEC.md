## TECHNICAL REQUIREMENTS

### Stack
- React 18+ with TypeScript
- Vite for bundling
- Framer Motion for animations
- GSAP for scroll-triggered animations
- Three.js or React Three Fiber for 3D elements (optional but impressive)
- Tailwind CSS for styling
- Particles.js or tsParticles for particle effects

### Features Required
1. **Preloader** - Cinematic, like Shopify Editions
2. **Smooth scroll** - Lenis or similar
3. **Scroll-triggered animations** - GSAP ScrollTrigger
4. **Parallax effects** - Depth and movement
5. **Cursor effects** - Custom cursor with interactions
6. **Page transitions** - Smooth, branded transitions
7. **Responsive** - Mobile-first, works everywhere
